Floating point numbers

This is a worked example showing how a floating point number is stored. This example uses a 32-bit float. Smaller 16-bit and 8-bit floats use the same idea: split the bits into sign, exponent, and fraction fields.

Appendix

Other float sizes

32-bit float

1 sign bit, 8 exponent bits, 23 fraction bits. This is the format used in the worked example.

16-bit float

1 sign bit, 5 exponent bits, 10 fraction bits. This is commonly called half precision.

8-bit float

There is no single universal 8-bit float. A common teaching format is 1 sign bit, 4 exponent bits, and 3 fraction bits.
